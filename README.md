# あとで読む（読まない）

Webページ上のリンクを右クリックから素早くメモと共に保存できるChrome拡張機能。  
**保存して満足するタイプのあなたへ。**

![Extension Icon](src/icons/icon128.png)

## 🌟 特徴

- **シンプル**: リンク上で右クリック → メモ入力 → 保存
- **編集可能**: URLは修正可能（リダイレクトURL対応）
- **Markdown形式**: 汎用性の高い形式で保存
- **ローカル保存**: データはあなたのPCに安全に保存
- **キーボードショートカット**: `Ctrl+Enter`で保存、`Esc`でキャンセル
- **読まなくてもOK**: 保存した瞬間が一番幸せ

## 📦 インストール

1. このリポジトリをクローンまたはダウンロード
2. Chromeで `chrome://extensions/` を開く
3. 「デベロッパーモード」を有効化
4. 「パッケージ化されていない拡張機能を読み込む」をクリック
5. ダウンロードした**`src`フォルダ**を選択

詳しいインストール手順は [docs/user_guide.md](docs/user_guide.md) を参照してください。

## 🚀 使い方

1. 拡張機能アイコンをクリックして保存先MDファイルを設定
2. リンク上で右クリック
3. 「リンクをメモに保存」を選択
4. メモを入力して保存

## 📝 保存形式

```markdown
## 2026-02-07 17:30:45
- URL: https://example.com
- メモ: あとで読む

---
```
