# あとで読む（読まない）

Webページを右クリックから素早くメモと共に保存できるChrome拡張機能。

![Extension Version](https://img.shields.io/badge/version-1.0.0-blue.svg)
![License](https://img.shields.io/badge/license-MIT-green.svg)

## 特徴

### 3種類の保存方法

- **リンクを保存**: リンク上で右クリック → 「リンクをメモに保存」
- **選択範囲から保存**: テキスト選択 → 右クリック → 「選択範囲からURLを保存」
- **ページを保存**: ページ本体で右クリック → 「このページを保存」

### 充実の管理機能

- **一覧表示**: 保存したメモをカード形式で表示
- **編集・削除**: 保存済みメモの編集・削除
- **タグ機能**: メモにタグを付けて分類
- **検索機能**: URL・メモ・タグから検索

### その他の特徴

- **シンプル**: 直感的なUI・操作
- **Markdown形式**: 汎用性の高い形式で保存
- **ローカル保存**: データはあなたのPCに安全に保存
- **モダンなUI**: ダークテーマ、トースト通知

## インストール

1. このリポジトリをクローンまたはダウンロード

   ```bash
   git clone https://github.com/kaguya25/read-later-not.git
   ```

2. Chromeで `chrome://extensions/` を開く

3. 「デベロッパーモード」を有効化

4. 「パッケージ化されていない拡張機能を読み込む」をクリック

5. ダウンロードした**`src`フォルダ**を選択

詳しいインストール手順は [docs/user_guide.md](docs/user_guide.md) を参照してください。

## 使い方

### 初回セットアップ

1. 拡張機能アイコンをクリック
2. 保存先MDファイルを新規作成または選択
3. 完了

### リンクを保存

1. リンク上で右クリック
2. 「リンクをメモに保存」を選択
3. メモとタグを入力して保存

### 選択範囲から保存

1. URLを含むテキストやリンクテキストを選択
2. 右クリック → 「選択範囲からURLを保存」
3. 自動抽出されたURLを確認して保存

### 現在のページを保存

1. ページ上（リンク以外）で右クリック
2. 「このページを保存」を選択
3. ページタイトルが自動入力されるので、確認して保存

### 保存したメモを見る

1. 拡張機能アイコンをクリック
2. 一覧画面で検索・閲覧・編集・削除

## 保存形式

Markdownファイルに以下の形式で保存されます：

```markdown
## 2026-02-07 17:30:45
- URL: https://example.com
- メモ: あとで読む
- タグ: [技術, Chrome拡張, JavaScript]

---
```

## 主な機能

| 機能 | 説明 |
|------|------|
| リンク保存 | リンク右クリックで保存 |
| 選択範囲保存 | テキストからURL自動抽出（リンク要素+正規表現） |

| ダークテーマ | 目に優しいモダンなUI |

### メモ入力画面

ダークテーマのモダンなUI。URL・メモ・タグを入力して保存。

保存したメモをカード形式で表示。検索・編集・削除が可能。

## アイコンについて

適切なサイズのアイコン（16px、48px、128px）を設定してください。
プロジェクトに含まれるSVGファイルから生成できます。

## よくある質問

**Q: データはどこに保存されますか？**  
A: ユーザーが選択したローカルのMarkdownファイルに保存されます。

**Q: オフラインで使えますか？**  
A: はい、完全にオフラインで動作します。

**Q: 他のブラウザでも使えますか？**  
A: Chromium系ブラウザ（Edge、Brave等）で動作します。

**Q: データを他のPCに移行できますか？**  
A: Markdownファイルをコピーするだけで移行できます。
